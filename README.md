# GptMahjong
把GPT3.5当做一个玩家，从所有可能的打出的手牌中选择一张打出，和麻将算法进行对战。
需要修改AI文件中human类中的get_move_from_prompt函数中的apikey
麻将的游戏环境来自https://github.com/Haskely/Mahjong
